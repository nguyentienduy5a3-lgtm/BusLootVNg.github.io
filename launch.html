BusLoot — Launch

Nhấp nút bên dưới để mở prototype trong tab mới (index.html).

Nếu trình duyệt chặn popup hoặc cấm mở file cục bộ, xem hướng dẫn server nhanh.

Hướng dẫn server nhanh 
# Mở PowerShell trong thư mục D:\BusLoot
python -m http.server 8000
# Sau đó mở trình duyệt: http://localhost:8000/
